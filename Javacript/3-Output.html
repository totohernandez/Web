Output

JavaScript can display data in different ways:

    Writting into an HTML element, using innerHTML
    Writting into the HTML output using document.write()
    Writting into an alert box, using window.alert()
    Writting into the browser console, using console.log()

Using console.log()

JavaScript Print

JS does not have any print object or print method.

You cannot acess output devices from JS.

The only exception is that you can call the window.print() method in the browser to print the content of the current window.
