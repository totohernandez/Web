Form Structure

Form controls live insde a form element.

Every form elements requires an action attribute. Its value is the URL for the page on the server that will recive the information when the form is submitted.

Forms can be sent using one of two methods: get or post.

Get: the values from the form are added to the end of the url specified on the action attribute. Used to get short forms, and retrieving data from the web server.

Post: the values are sent in what are known as HTTP headers. Used to allow users to upload a file, sensitive data and to add or delete information from a database.

This is where the form controls will apear.
